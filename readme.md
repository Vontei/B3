#B3
###Backcountry Blue Book
This application is an avalanche observation tool designed to collect and disseminate backcountry field observations according to the AIARE
standards.

The app is built with:
* Javascript
* MongoDb
* Monk
* Node.js
* Jade Templating
* Express Framework
* BcryptJs Encryption
* cookie-session
